Juego de Piedra, Papel o Tijera para parejas con castigos atrevidos, modo multijugador local y versión móvil instalable como app.

## 🎮 [Jugar ahora](https://jh4m1l.github.io/primer_repositorio/index-movil.html)

## ¿Qué incluye?
- 💋 4 niveles de intensidad
- 📱 Pantalla dividida para 2 jugadores en un celular
- 📄 Castigos editables desde un archivo JSON
- ⚡ Instalable como app (PWA)
- 🎉 Confetti, animaciones y modo libre

## Tecnologías usadas
- HTML
- CSS
- JavaScript
- JSON

## Cómo usarlo
1. Abrí el link de arriba desde tu celular
2. Elegí el nivel de calor
3. Pongan los nombres
4. ¡Jueguen!
